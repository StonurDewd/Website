Server Commands

/TPS Shows the Server Ticks Per Second @ 1m, 5m, 15m.
/MSG Sends a message to desired user. (/msg username msg).
/Tell Sends a message to desired user. (/tell username msg).
/R Sends a message to desired user. (/r msg).
/Ignore Ignores the desired user. (/gnore user).
/MOTD Displays the ONJOIN MOTD.
/Spawn Brings you to Server Spawn.
/AACH Displays the avialable codes for Advanced Achievements.
